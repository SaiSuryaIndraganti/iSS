# iSS: The Digital Happiness Platform

> A premium, gamified digital space designed to satisfy late-night shopping impulses, track financial goals, and curate mindfulness—without spending a single rupee.

## What the project does?
iSS intercepts the psychological loop of "doom-shopping." It provides a beautifully designed, functional e-commerce interface where users can browse high-end curated products, create custom items ("Dreams"), and complete a virtual checkout process. Instead of charging money, the platform calculates the exact amount the user *didn't* spend and adds it to a visual "Dream Fund" tracker.

## Why the project is useful?
Consumer psychology often drives us to scroll shopping apps when we are bored, anxious, or unable to sleep. We want the dopamine hit of discovery and collection, not the buyer's remorse that follows. 
iSS provides that exact emotional loop in a safe, calming environment. It stands out by offering:
* **The Dream Fund Dashboard:** Turns the act of *not* spending into a visible financial victory toward a real-world goal.
* **Mood-Based Browsing:** Filters content based on emotional states (Cozy, Focus, Creative) rather than just utility.
* **Custom Dream Builder:** Allows users to manually input, price, and save items they find in real life to track their savings toward them.
* **Healthy Gamification:** Rewards users with XP, daily streaks, and milestone achievements for demonstrating mindful self-control.

## How users can get started with the project?
This project is currently in its fully functional HTML Prototype phase (v0.22.0). 
1. Clone or download this repository.
2. Open the `iSS.html` file in any modern mobile or desktop web browser.
3. No external databases, servers, or installations are required. All state management, gamification, and custom dreams run locally and persistently on your device via the browser's Storage Engine.
